## Key Differences

### AI Agent Integration
Patent Search MCP is built for AI agents using the MCP protocol. AI agents call tools like `search_patent(number="US10123456")` or `get_patent_citations(number="US10123456")` directly — no manual patent database navigation.

Google Patents and USPTO are web-based tools designed for human researchers. They have no programmatic API for AI agent integration.

### Multi-Jurisdiction Coverage
Patent Search MCP covers USPTO, EPO, and Google Patents in a single workflow. Google Patents covers multiple jurisdictions but is a web UI. USPTO only covers US patents.

### Use Case Focus
For AI agents doing freedom-to-operate analysis, patent landscaping, or IP due diligence — programmatic access matters more than free pricing. At $0.05/call, 2,000 patent lookups costs $100 vs. weeks of manual research time.

## When to Choose Each

**Choose Patent Search MCP when:**
- You're an AI agent doing patent research
- You need programmatic access for workflows
- You need multi-jurisdiction patent data

**Choose Google Patents when:**

- You need a free web interface

**Choose USPTO when:**
